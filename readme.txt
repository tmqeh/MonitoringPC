코딩 규칙

try except finally 로직에 포함 시키기 (finally는 선택적)

get : 단건
list : 배열
write : 파일 쓰기
check : Y/N 표시 또는 1byte return 값

함수 종료 후에는 개행을 2줄로 한다.
띄워쓰기에서 tab은 금지한다.
들여쓰기 공백은 4개로 통일한다. "    "
간단한 조건문이라도 줄을 나눠 명확히 구분한다.

variables와 function은 "_"를 이용하여 snake case를 사용
constant는 항상 대문자를 사용 ("_" 포함 가능)

class는 첫글자가 대문자로 시작되는 calmel case 즉, pascal case를 사용한다.

import를 할때는 한줄에 하나의 라이브러리만 호출한다. 
import문은 항상 최상단에 위치한다.
import의 대상은 항상 절대경로를 표시한다. 
※ 절대경로 : C:\...\...가 아니라 sys.path에 포함된 경로의 최상위부터 인지
※ 상대경로 : 실행하는 *.py 파일의 위치에서 부터 표기

큰따옴표 ("") : 통상적으로 사용
작은따옴표 ('') : 식별자에 사용 ex) dict의 key값
           또는 큰따옴표로 묶인 string
=> 꼭 지킬 필요는 없지만 지향하자
